\section{Cross-model size sweep}
\label{app:cross-model}

\paragraph{Methods.}
The cross-model study repeats the baseline readouts across five instruction-tuned
models---Qwen3 (0.6\,B, 4\,B), Llama-3.2 (1\,B, 3\,B), and
Mistral-7B-Instruct-v0.3---and plots accuracy against log model size, with
$n{=}770$ per metric cell per model. No scaffold and no surface variation are
applied.

\paragraph{Results.}
\begin{itemize}[leftmargin=*,topsep=2pt,itemsep=1pt]
  \item Three-option abstention rises with scale within each family: Qwen $0.774$
    (0.6\,B) $\to 0.892$ (4\,B); Llama $0.299$ (1\,B) $\to 0.887$ (3\,B).
  \item Greedy-thinking abstention rises likewise: Qwen $0.534 \to 0.718$; Llama
    $0.275 \to 0.834$.
  \item \texttt{Llama-3.2-1B} abstains least on the three-option readout ($0.299$,
    below the $1/3$ chance line) and has the highest disambiguated-target rate
    ($0.678$ vs.\ $0.395$ for \texttt{Qwen3-0.6B}).
  \item Two-option disambiguated accuracy (abstention $0$ by construction),
    sub-1.3\,B: Qwen-0.6B $0.596$, Llama-1B $0.587$.
  \item Two-option disambiguated accuracy, 3--7\,B: Llama-3B $0.822$, Qwen-4B
    $0.882$, Mistral-7B $0.835$.
\end{itemize}

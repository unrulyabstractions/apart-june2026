\section{Dynamics: forking-paths}
\label{app:dynamics}

\paragraph{Methods.}
Following the forking-paths method, we fork the greedy reasoning path of one
representative ambiguous item at \emph{every} token position along the full chain
of thought, parse each continuation to an outcome, and track the outcome
distribution $O_t$ along the path. A Bayesian change-point posterior
$p(\tau{=}t\mid y)$ tests for a single commitment token; a per-position diversity
$H(O_t)$ (Shannon entropy of the live answers) measures how many answers are still
in play; and a branching tree summarizes the root $\to$ trunk $\to$ identity-branch
outcome mix. Every forked position retains its raw rollout generations, so the
outcome at each step is auditable.

\paragraph{Results.}
\begin{itemize}[leftmargin=*,topsep=2pt,itemsep=1pt]
  \item On \texttt{Qwen3-14B} the outcome distribution $O_t$ stays pinned on
    \emph{abstention} for the first ${\sim}260$ of $351$ chain-of-thought tokens,
    then swings sharply near the end of reasoning. The Bayesian change-point test
    finds a \emph{significant} single commitment token at $t{=}270$ (Bayes factor
    ${\approx}10^{9}$, $p(m{\ge}1){=}1.0$): the larger model's answer snaps at one
    deciding token rather than accreting.
  \item This sharpens with scale. The smallest model (\texttt{Qwen3-0.6B}) shows
    \emph{no} significant commitment token over the same test (Bayes factor
    $0.023$, $p(m{=}0){=}0.98$)---its answer accretes gradually---so the
    forking-paths signature of a single deciding token emerges only at the larger
    size.
  \item The most-forking position (where re-sampling a different token most diverts
    the outcome) is $t{=}344$, late in the chain; diversity $H(O_t)$ sits near zero
    through the abstention plateau and spikes only across the terminal swing.
\end{itemize}

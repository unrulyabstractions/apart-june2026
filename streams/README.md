# Research Streams

Parallel research threads on the temporal-preference manifold. Each stream has
its own folder for notes, experiments, and results.

- `stability_generalization/` — Stability and generalizability of the manifold:
  does the same structure persist across scenarios, phrasings, seeds, and models?
- `prediction/` — Predictions through the manifold: using manifold coordinates
  to predict behavior (choices, discounting) on held-out inputs.
- `intervention/` — Interventions in the manifold: steering and patching along
  manifold directions to causally change model behavior.
- `evolution/` — Evolution of the manifold: how the structure emerges and
  changes across layers, token positions, training checkpoints, and model scale.
